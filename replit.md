# Tech Savy Tonydave Portfolio

## Overview
A static HTML/CSS/JS personal portfolio website showcasing frontend development work.

## Project Structure
- `index.html` - Main landing page
- `project.html` - Projects showcase page
- `contact.html` - Contact page
- `style.css` - Main stylesheet
- `index.js` - JavaScript for animations and carousel functionality
- `media/` - Images and icons

## Running the Project
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment serving from the root directory.
